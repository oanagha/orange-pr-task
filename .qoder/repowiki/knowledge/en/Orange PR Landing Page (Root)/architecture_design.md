Single-page static site rooted at `index.html`, which loads global styles from `css/style.css` and `css/responsive.css`, a reusable hero component stylesheet from `components/hero/hero.css`, and behaviour scripts from `components/hero/hero.js` plus `js/script.js`. The page is structured as a full-viewport hero section containing a header (logo + nav + mobile menu icon), a three-slide carousel (`hero__slides`) each pairing headline text with an autoplay muted loop video card, and prev/next dot controls. A second, minimal duplicate `<html>` block at the bottom of `index.html` serves as a quick smoke-test shell referencing only the base CSS/JS. Assets are organised under `assets/images`, `assets/videos`, and `assets/icons`; there is no build toolchain or server — the project is intended to be served directly by any static file host.
